# Wharton Hockey Project (Round 1)

This github repository contains the notebooks used for exploratory analysis, feature engineering, modeling, and visualizations.

### 2) `hockeyModelMAIN`

### 3) `Offense1v2Disp`

### 4) `HockeyVisualizations`
  - Notebook for visualizing the correlation between power ranking and offensive team disparity.
