The objective of the Wharton Data Science competition: We are given raw hockey data and our task is to create:
  1. A power ranking of the 32 teams 
  2. A match prediction for the 1st round of the playoffs
  3. An analysis between the quality of the first and second teams. 

I have attached two jupyter notebook files. 

- HockeyEDA.ipynb (Explroatory Data Analysis) is for the first part of the project which is creating a power ranking. In this file I have also created many intermediate datasets
- hockeyModel.ipynb is for creating model theat predicts the winner of the 1st round of the playoffs.
